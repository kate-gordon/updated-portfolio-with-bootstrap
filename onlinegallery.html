Online Gallery
Georgia Artists with DisAbilities
github code

01.

Brainstorming

As a team, we reached out to the organization, Georgia Artists with DisAbilities, to propose building an online gallery website. They sent us the artwork and artist information.

02.

Designing & Planning

I led the design and front end implementation. Using Figma I created wireframes demonstrating our MVP (An Online gallery that displays artist's work, artist's bio pages, and a checkout option to purchase the work. This process helped organize components and acted as a blueprint for the build of our React application.

03.

Building

With one other teammate, I built the architecture and the functionality of the components in our app over a 4 day period. We built using React, Context API for state management, Stripe for our cart, and Chakra UI for our design system.
